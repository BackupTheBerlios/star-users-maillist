"Felix E. Klee" <felix.klee at inka.de> wrote:

> It seems that "star" incrementally backups entire files, if only their
> ctime changed. Is this observation correct?

This is correct. Note that star could backup much less in case that
we did forbid to change the mtime of files via utimes(2)...e.g. extracting tar archives.

If you like to experiment with highly dangerous stuff, use "-dumpmeta".
Note that this is definitely not recommended!
