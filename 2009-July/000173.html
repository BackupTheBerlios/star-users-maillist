Thank you for the hint, this was caused by a wrong symlink follow flag in the 
"install source" program (-h instead of -L):

Usage: NEWSRC [-h] [-L] fromdir todir
Options:
        -r      Recursive copy

        -h      Do not follow symbolic Links on create side
                Needed when the source dir uses symlinks
                that should not become files on the target side.

        -L      Follow symbolic Links on extract side
                Needed when the source dir does not use symlinks
                but the target dir contains symlinks.

        -U      Extract files unconditionally
