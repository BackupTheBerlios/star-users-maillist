This would be great.

The point is that a script might rely on star's return code to 
decide whether two directory trees are identical. If one of these 
directories exists and the other one not, they are certainly not 
identical. Of course, one could add a test like 'test -d ...' to 
the script, but I think that the cleaner solution would be to let 
star do all the work.
